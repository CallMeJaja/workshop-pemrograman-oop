# Latihan Level 1: Class & Object Dasar

## Deskripsi

Latihan ini bertujuan untuk memahami dasar pembuatan class, atribut, objek, dan penggunaan static variable.

## Daftar Soal

### Soal 1: Class Buku

Buatlah class `Buku` dengan atribut:

- `judul` (String)
- `pengarang` (String)
- `harga` (int)

Buat 3 objek buku berbeda dan tampilkan datanya.

---

### Soal 2: Class Mahasiswa dengan Static Variable

Buat class `Mahasiswa` dengan:

- Atribut instance: `nama` (String), `nim` (String), `ipk` (double)
- Atribut static: `jumlahMahasiswa` (int)

Setiap kali objek dibuat, `jumlahMahasiswa` bertambah. Buat 3 objek dan tampilkan jumlah total mahasiswa.

---

**Kerjakan soal di kertas dan pastikan memahami struktur class, deklarasi atribut, serta konsep instance dan static variable.**
