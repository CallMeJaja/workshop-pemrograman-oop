# Latihan Persiapan Ujian Pemrograman Berorientasi Objek (PBO)

## Deskripsi

Repositori ini berisi latihan-latihan dan studi kasus sebagai **persiapan Ujian PBO Paper Based**. Semua soal dirancang sesuai dengan materi RPS dan pola soal yang sering keluar seperti pada laporan praktikum sebelumnya.

Materi latihan:

- Class & Object (Dasar)
- Constructor & Method (Termasuk Overloading)
- Enkapsulasi, Getter/Setter, Validasi
- Studi kasus terintegrasi (sistem sederhana dengan logika bisnis)
- Bonus: Tips pengerjaan dan checklist

## Struktur Folder/Isi File

```
/Class-Object/          # Latihan Dasar Class & Object
/Constructor-Method/    # Latihan Constructor & Method
/Enkasulasi-Validasi/   # Latihan Enkapsulasi & Validation
/Studi-Kasus/           # Studi Kasus Terintegrasi
README.md               # File dokumentasi ini
```

- Setiap folder berisi soal dan contoh kerangka jawaban (bisa kamu kerjakan di kertas atau salin ke IDE untuk latihan).
- Gunakan README ini untuk referensi ringkas dan panduan latihan.

## Petunjuk Penggunaan

1. **Baca roadmap latihan pada README**  
   Mulai dari Level 1 (paling dasar), lanjutkan ke level berikutnya setelah menguasai.
2. **Kerjakan setiap soal di kertas**  
   Simulasi kondisi ujian asli, TIDAK boleh copy-paste langsung dari komputer/IDE.
3. **Cek dengan flow mental/trace table**  
   Pastikan logika program sudah benar meski tanpa dijalankan di komputer.
4. **Gunakan file laprak sebelumnya sebagai referensi**  
   Cocokkan pola soal dengan laporan praktikum yang sudah kamu buat.

## Tools/Bahasa yang Digunakan

- Bahasa utama: **Java (PBO Dasar)**
- Dapat dikembangkan dalam VB.NET sesuai kebutuhan modul dan preferensi kelas.
- Tools latihan: Kertas/bolpoin (simulasi ujian), IDE hanya untuk validasi logika jika diperlukan.

## Tips & Best Practice

- Fokus pada **struktur program**: class → atribut → constructor → method → main
- Jangan lupa tuliskan **akses modifier**, penggunaan **this**, dan validasi pada setter/method.
- Gunakan **komentar** untuk menjelaskan jika waktu pengerjaan terbatas.

## Checklist Persiapan

- [ ] Faham cara mendesain class, atribut, dan method
- [ ] Bisa membuat constructor dengan overloading
- [ ] Paham enkapsulasi, getter, setter, dan validasi
- [ ] Bisa mengerjakan studi kasus terintegrasi (logika bisnis)
- [ ] Terbiasa menulis kode di kertas tanpa IDE
