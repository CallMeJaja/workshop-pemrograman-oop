# Latihan Level 4: Studi Kasus Terintegrasi

## Deskripsi

Level ini berisi soal studi kasus terintegrasi yang melibatkan banyak konsep sekaligus: class, method, constructor, enkapsulasi, logic percabangan, perulangan, dan integrasi logika bisnis.

## Daftar Soal

### Soal 7: Sistem Rental Mobil

Buat program rental mobil menggunakan class `Mobil` dengan fitur:

- Data mobil (merk, tipe, harga, status tersedia)
- Perhitungan biaya rental dan diskon
- Fitur rental dan pengembalian
- Simulasi transaksi antar objek

### Soal 8: Sistem Pengelolaan Inventori Toko

Buat program inventori barang toko dengan class `Barang`:

- Tambah/kurang stok, perhitungan nilai inventori, cek stok minimum, validasi stok saat penjualan

### Soal 9: Sistem Kalkulator Nilai Mahasiswa

Buat program kalkulator nilai mahasiswa:

- Input nilai tugas, UTS, UAS
- Hitung nilai akhir, huruf mutu, dan keterangan kelulusan
- Tampilkan transkrip lengkap untuk beberapa mahasiswa

---

_Kerjakan dengan urutan: desain class, atribut, constructor, method, lalu implementasi simulasi pada main program. Fokus pada integrasi logika dan praktik penulisan kode di kertas._
