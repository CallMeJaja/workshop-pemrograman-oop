# Latihan Level 2: Constructor & Method

## Deskripsi

Pada level ini, latihan fokus pada pembuatan constructor (termasuk overloading) serta method (dengan atau tanpa return value).

## Daftar Soal

### Soal 3: Class Karyawan dengan Constructor

Buat class `Karyawan` dengan:

- Atribut: `nama` (String), `jabatan` (String), `gaji` (int)
- Constructor untuk inisialisasi semua atribut
- Method `tampilkanInfo()` untuk menampilkan data karyawan

### Soal 4: Overloaded Constructor pada Produk

Buat class `Produk` dengan:

- Atribut: `kode` (String), `nama` (String), `harga` (int), `stok` (int)
- Constructor 1: hanya `nama`
- Constructor 2: `nama` dan `harga`
- Constructor 3: semua atribut
- Method `tampilkanProduk()`

Buat 3 objek dengan constructor berbeda.
